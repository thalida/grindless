Config¶

These global constants are used by the build and release scripts. Learn more about scripts.

Quick Reference¶

DIST_DIR

Output directory of the built datapack.

DIST_REGION_FNS_DIR

Output directory of the built regions mcfunctions.

REGIONS_DIR

Directory with all of the region specific settings.

REGION_TEMPLATE_PATH

Absolute path to the region template.

SCRIPTS_DIR

Holds all of the comandline-run scripts.

SETTINGS_DIR

Directory with all of the datapack settings.

SOURCE_DIR

Subdirectory which includes all of the core logic.

TEMPLATES_DIR

Directory with partial templates and methods.

VERSION

Datapack Version

ZIPS_DIR

Output directory of the zipped datapack build.

Globals¶

grindless.config.DIST_DIR = /abs/path/to/grindless-repo/build[source]¶

Output directory of the built datapack. This folder is gitignored.

Used by: build script and release script

grindless.config.DIST_REGION_FNS_DIR = <DIST_DIR>/data/grindless/functions/regions[source]¶

Output directory of the built regions mcfunctions.

Used by: build script

grindless.config.REGIONS_DIR = <SETTINGS_DIR>/regions[source]¶

Directory with all of the region specific settings. Learn more about regions.

Used by: build script

grindless.config.REGION_TEMPLATE_PATH = <TEMPLATES_DIR>/region.jinja[source]¶

Absolute path to the region template. Learn more about templates.

Used by: build script

grindless.config.SCRIPTS_DIR = <SOURCE_DIR>/scripts[source]¶

Holds all of the comandline-run scripts.

Used by: build script

grindless.config.SETTINGS_DIR = <SOURCE_DIR>/settings[source]¶

Directory with all of the datapack settings.

Used by: build script and settings

grindless.config.SOURCE_DIR = /abs/path/to/grindless-repo/grindless[source]¶

Subdirectory which includes all of the core logic.

Used by: build script and release script

grindless.config.TEMPLATES_DIR = <SOURCE_DIR>/datapacks/grindless/templates[source]¶

Directory with partial templates and methods. Learn more about templates.

Used by: build script

grindless.config.VERSION = '1.0.0'[source]¶

Datapack Version

Used by: release script

grindless.config.ZIPS_DIR = /abs/path/to/grindless-repo/releases[source]¶

Output directory of the zipped datapack build. This folder is gitignored.

Used by: release script
