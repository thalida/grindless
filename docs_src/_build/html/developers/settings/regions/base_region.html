Base Region¶

The base class extended by all regions. This class contains the logic for generating region configs, which are passed to the jinja templates during build.

Module Overview¶

Base class for all region config classes.

Parsed settings/enchantments.yaml file

Parsed settings/items.yaml file

tools_config

Parsed settings/tools.yaml file

static create_enchantment_damage_configs(tool_type, tool_option, damage_amount)[source]¶
Creates dictionary of where the key is the tools with enchantments

and the value is the original damage modified by the enchantments. For example, having an unbreaking enchantment will cause the tool to receive less damage.

Parameters

  • tool_type (str) – The base type of the tool (eg. pickaxe)

  • tool_option (str) – The current version of the tool (eg. default or silk_touch)

  • damage_amount (float) – How much damage is given by default to this tool

Returns

Dict of enchantment keys mapped to modified damage amount

Return type

dict

static create_enchantment_drops_configs(tool_option, give_amount)[source]¶
Creates a dictionary were the keys are the tools with enchantments

and the values are how much of an item should be given. For example, having an efficiency enchantment will cause the tool to give more items.

Parameters

  • tool_option (str) – What version of the tool are we modifying?

  • give_amount (float) – How much of the item is given by this tool already?

Returns

Dict of the enchantment keys mapped to the modified give amount

dict

static get_tool_info(tool)[source]¶

Get the tool type and material from a tool key

Parameters

Returns
